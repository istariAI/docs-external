<h2 className="text-2xl font-normal mt-10 mb-4 text-gray-900 dark:text-white" style={{ fontFamily: "Georgia, 'Times New Roman', serif" }}>Try the API</h2>

A single `POST /v2/search` call discovers companies by keyword, natural-language description, similarity to a known domain, or structured filters.

<CodeGroup>

```bash cURL
curl -X POST https://api.istari.ai/v2/search \
  -H "Authorization: Bearer $ISTARI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "describe": "B2B companies that manufacture industrial heat pumps",
    "filters": { "country": ["Germany"] },
    "columns": ["domain", "name", "country"],
    "size": 50
  }'
```

```python Python
import os, requests

resp = requests.post(
    "https://api.istari.ai/v2/search",
    headers={"Authorization": f"Bearer {os.environ['ISTARI_API_KEY']}"},
    json={
        "describe": "B2B companies that manufacture industrial heat pumps",
        "filters": {"country": ["Germany"]},
        "columns": ["domain", "name", "country"],
        "size": 50,
    },
)

for row in resp.json()["data"]:
    print(row["domain"], "—", row["name"])
```

```typescript TypeScript
const resp = await fetch("https://api.istari.ai/v2/search", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${process.env.ISTARI_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    describe: "B2B companies that manufacture industrial heat pumps",
    filters: { country: ["Germany"] },
    columns: ["domain", "name", "country"],
    size: 50,
  }),
});

const { data } = await resp.json();
data.forEach((row) => console.log(row.domain, "—", row.name));
```
