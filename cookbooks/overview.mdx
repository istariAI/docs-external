title: "Overview"
description: "GOI cookbooks turn fuzzy requests into clean, reproducible results with fixed parameter recipes."
---

**Cookbooks** are ready-made GOI workflows for common research tasks: supplier shortlists, lookalikes, market sizing, startup scouting, and more. Each one is a **query template** — a fixed parameter set where you supply one or two inputs and everything else stays locked in. Quality comes from the parameters, not the search string, so you get the same clean result every time without knowing the internals.

All examples in the query templates map the same parameters as in the [API](/api/api_start_page) `POST /v2/search`.
