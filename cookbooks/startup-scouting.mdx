---
title: "Startup scouting"
description: "Find early-stage organizations working on a specific theme, scoped to startups, for sourcing, partnerships, or trend mapping."
---

**Inputs:** `theme` (free text) · optional `location`. **Returns:** a ranked list of startups matching the theme.

Combine a semantic `query` (the theme) with a structured `organization_type=["Startup"]` filter so you get only early-stage organizations, not incumbents or consultancies.

## The recipe

```python
search_organizations(
    query=f"{theme}",
    organization_type=["Startup"],
    organization_size=["Micro (0-9)", "Small (10-49)"],  # optional, tighten to early-stage
    country=[country],                                    # optional scope
    size=20,
    dedup=True,
)
```

## What each lever does

| Lever | Role |
|---|---|
| `query` | The theme, in natural language: describe the problem space, not just a keyword. |
| `organization_type` = **Startup** | The key filter. Restricts to organizations typed as startups. |
| `organization_size` | Optional: narrows to early-stage headcount bands. |
| `min_score` | Optional. Raise to `~0.5` if the tail drifts off-theme (semantic relevance decays past the top results). |

## Validated example

`theme="climate tech building carbon accounting and emissions tracking software"`:

| Organization | Location |
|---|---|
| Cozero | Berlin, Germany |
| Atoma Research | United States |
| Sequest | United States |
| CATUDI | Italy |
| Climate Founders | Germany |

## Notes

- **Semantic relevance decays** down the list: the genuinely on-theme startups are the top ~10–30; the tail fills with looser matches. Use `size` modestly or raise `min_score`.
- **The startup-typed pool is small** for any given theme (this query had ~229 candidates), so cast the `query` a bit wider rather than over-narrow.
- **Don't add `nace_code`** unless the user names an industry: it can exclude relevant cross-sector startups. Let the semantic query do the topical work.
- For an exhaustive, unranked universe (full outreach list) over a *structured* slice, use [`filter_organizations`](/mcp/tools) instead of search.
